## 프로젝트 소개

우리는 감정을 억누르는 법을 배웁니다. 필브는 이러한 감정 억압을 해소하고 명확한 감정 인지를 돕는 것을 목표로 합니다. 감정 분류, 분석을 통해 사용자의 감정을 파악하고, 피드백을 제공하는 서비스를 개발했습니다.
